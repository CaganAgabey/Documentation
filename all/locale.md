# Locale
---
### Descripción
Este comando te permite configurar tu idioma en el bot. Usar este comando sin parámetros mostrará la lista de idiomas disponibles y el idioma configurado.

?> Este comando configura tu idioma a nivel de __usuario__. Este idioma se mostrará en las respuestas y mensajes que recibas del bot.
El idioma predeterminado es el inglés.

?> Si quieres ayudar a traducir el bot a tu idioma, mira la [página de traducciones](es/translation.md).
### Argumentos Opcionales
`idioma` - El idioma que quieres configurar.

### Uso
```
.locale  [idioma]
```
### Alias
`language`, `lang`, `translate`
### Permisos Requeridos
Cualquier usuario puede usar este comando.
