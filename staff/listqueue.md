# Listqueue
---
### Description
Cette commande permet d'afficher une liste des suggestions en attente d'approbation. 
### Utilisation
```
.listqueue
```
### Aliases
`queue`, `showqueue`
### Permission Requise
Toutes les personnes ayant la permission **Gérer le serveur**, un rôle admin configuré ou un rôle staff configuré.
